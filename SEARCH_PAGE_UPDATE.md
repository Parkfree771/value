# 검색 페이지 분리 및 모바일 피드 개선

## 개요

메인 피드 페이지를 모바일에 최적화하고, 검색 및 필터 기능을 별도 페이지로 분리했습니다.

## 주요 변경 사항

### 1. 메인 피드 페이지 (`app/page.tsx`)

#### 모바일 버전 개선
- **검색/필터 제거**: 모바일에서 SearchBar와 FilterBar 숨김
- **검색 버튼 추가**: `/search` 페이지로 이동하는 버튼 배치
- **4개 탭 그리드**: 2x2 그리드 레이아웃으로 깔끔하게 정리

```tsx
// 모바일 탭 구조 (2x2 그리드)
📋 전체 피드    👥 팔로우
🔥 인기순      📈 수익률순
```

#### 데스크탑 버전
- **검색/필터 유지**: 기존 SearchBar와 FilterBar 표시
- **4개 탭 가로 배치**: 전체 피드, 팔로우 피드, 인기순, 수익률순

### 2. 검색 전용 페이지 (`app/search/page.tsx`)

새로 생성된 검색 전용 페이지의 주요 기능:

#### 헤더
- **뒤로가기 버튼**: 모바일에서 이전 페이지로 돌아가기
- **페이지 제목**: "검색"

#### 검색 기능
- **SearchBar**: 자동완성 지원
- **FilterBar**: 기간, 시장, 의견, 정렬 필터
- **검색 결과 요약**: "검색어" 검색 결과 N개

#### 검색 상태별 UI

**1. 초기 상태 (검색어 없음)**
```
🔍 검색어를 입력해주세요
종목명, 티커, 작성자로 검색할 수 있습니다

[인기 검색어]
1 삼성전자  2 NVIDIA  3 Apple ...
```

**2. 검색 결과 있음**
```
"삼성전자" 검색 결과 3개

[리포트 카드 목록]
[더 보기 버튼]
```

**3. 검색 결과 없음**
```
🔍 검색 결과가 없습니다
다른 검색어를 입력해보세요
```

#### 인기 검색어
- 8개의 인기 검색어 버튼
- 클릭 시 해당 키워드로 자동 검색
- 순위 번호 표시

### 3. 네비게이션 업데이트 (`components/Navbar.tsx`)

#### 데스크탑 메뉴
```
피드 | 검색 | 랭킹 | 글쓰기 | 마이페이지
```

#### 모바일 햄버거 메뉴
```
피드
검색  ← 새로 추가
랭킹
글쓰기
마이페이지
───────
로그인
회원가입
```

## UI/UX 개선 포인트

### 모바일 첫 화면 최적화

**Before (개선 전):**
```
[TOP 10 슬라이더]
[검색바]
[필터바 - 4개 필터 여러 줄로 표시]
[전체 피드] [팔로우 피드]
[리포트 목록]
```

**After (개선 후):**
```
[TOP 10 슬라이더]
[검색 버튼 - 클릭 시 검색 페이지로 이동]
[📋 전체] [👥 팔로우]
[🔥 인기] [📈 수익률]
[리포트 목록]
```

### 장점

1. **화면 공간 절약**
   - 검색/필터 기능 분리로 첫 화면이 깔끔해짐
   - 스크롤 없이 더 많은 리포트 확인 가능

2. **직관적인 탭 구조**
   - 이모지로 각 탭의 의미 명확히 전달
   - 2x2 그리드로 터치하기 쉬운 크기

3. **전용 검색 환경**
   - 검색에 집중할 수 있는 별도 페이지
   - 인기 검색어로 빠른 탐색 가능
   - 필터를 자유롭게 조합 가능

4. **일관된 UX**
   - 모바일 앱과 유사한 패턴
   - 뒤로가기 버튼으로 자연스러운 네비게이션

## 파일 구조

```
app/
├── page.tsx                 # 메인 피드 (검색/필터 모바일 제거)
└── search/
    └── page.tsx            # 새로운 검색 전용 페이지

components/
├── Navbar.tsx              # 검색 링크 추가
├── SearchBar.tsx           # 검색바 (재사용)
├── FilterBar.tsx           # 필터바 (재사용)
└── ReportCard.tsx          # 리포트 카드 (재사용)
```

## 반응형 동작

### 모바일 (< 768px)
- 메인: 검색 버튼 + 4개 탭 그리드
- 검색 페이지: 전체 기능 제공

### 데스크탑 (>= 768px)
- 메인: 검색바 + 필터바 + 4개 탭 가로 배치
- 검색 페이지: 동일한 기능 (별도 접근 가능)

## 향후 개선 사항

1. **검색 기록**: 최근 검색어 저장 및 표시
2. **필터 프리셋**: 자주 사용하는 필터 조합 저장
3. **음성 검색**: 모바일에서 음성으로 검색
4. **고급 검색**: AND/OR 조건, 범위 검색 등
5. **검색 추천**: AI 기반 연관 검색어 제안
6. **검색 결과 정렬**: 관련도순, 최신순 등 다양한 정렬

  ))}
